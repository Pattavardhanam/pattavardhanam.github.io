Custom 4-layer PCB for real-time ECG acquisition and arrhythmia detection, reducing
clinical leads to 2–3 using the Einthoven triangle.

- Designed analog front-end using TI ADS1298 (8-channel, 24-bit ECG AFE) with custom
  signal conditioning for real-time arrhythmia detection
- Developed 4-layer PCB in Altium integrating ADS1298, STM32F411 (Cortex-M4), and BLE
  module for wireless data streaming
